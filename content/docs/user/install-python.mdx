DeepTrade 框架依赖 Python ≥ 3.11。如果你还没有安装 Python，请参考以下指南。

## Windows

推荐使用官方安装包：

1. 访问 [Python 官方下载页面](https://www.python.org/downloads/windows/)。
2. 下载最新的 Python 3.11 或更高版本的 Windows installer (64-bit)。
3. **关键步骤**：在安装界面的底部，**务必勾选 "Add python.exe to PATH"**。
4. 点击 "Install Now" 完成安装。

## macOS

推荐使用 [Homebrew](https://brew.sh/) 安装：

<CopyableCommand command="brew install python" />

## Linux (Ubuntu / Debian)

通常系统自带 Python，如果版本低于 3.11，可以通过 `apt` 更新：

<CopyableCommand command="sudo apt update && sudo apt install python3 python3-pip" />

## 验证安装

打开终端或命令行，运行：

<CopyableCommand command="python --version" />

或

<CopyableCommand command="python3 --version" />

确保输出的版本号大于或等于 3.11。

## 下一步

Python 环境准备好后，即可 [安装 DeepTrade 框架](/docs/user/install)。
